# **Название проекта:** "Анализ продаж игр и рекомендации для формирования плана затрат на маркетинг" 

##  **Задача проекта:**
На основании ретроспективных данных о продажах игр, необходимо дать рекомендации по планированию затрат на продвижение игр в следующем году

##  **Стек проекта:**
- Python, 
- Pandas, 
- Matplotlib, 
- Plotly, 
- Seaborn, 
- Scipy

## **Ключевой результат проекта:**
- В рамках проекта был проведен анализ ретроспективных показателей продаж игр и даны рекомендации по формированию затрат на маркетинг на следующий год
## **Статус проекта:**
- Закончен (необходимо провести ревизию и актуализацию до 31.12.2022)
